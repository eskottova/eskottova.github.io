HTML

Wir werden heute in der Programmiersprache HTML (Hyper Text Markup Language) programmieren.

HTML Elemente

Elemente fangen mit einem Anfangstag an und hören meistens mit einem Endtag auf. <Beispiel>Text<Beispiel/>

Aufbau eines HTML Programms

<!DOCTYPE html>
<html>
 <body>
 </body>
</html>

Diesen Teil kann man als Anfänger eifach kopieren oder abtippen. Alles, was wir jetzt lernen werden, kommt zwischen <body> und </body>.

H Elemente

H ist die Abkürzung des englischen Wortes heading (Überschrift). Es gibt sechs verschiedene Grössen von Überschriften. Beispiel:

<body>
 <h1>Das ist die grösste Überschrift </h1>
 <h6>Das ist die kleinste Überschrift </h6>
</body>

Auf einer Website sähe das so aus:

Das ist die grösste Überschrift

Das ist die kleinste Überschrift

P Elemente

P ist die Abkürzung des englischen Wortes paragraph (Abschnitt/Absatz). Ps werden für alle Arten von Texen verwendet. Beispiel:

<body>
 <p>Das ist ein Absatz. </p>
 <p>Das ist ein anderer Absatz. </p>
</body>

Auf einer Website sähe das so aus:

Das ist ein Absatz.

Das ist ein anderer Absatz.
