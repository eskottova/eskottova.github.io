HTML

Wir werden in der Programmiersprache HTML (Hyper Text Markup Language) programmieren.



HTML Elemente

Elemente fangen mit einem Anfangstag an und hören meistens mit einem Endtag auf. <Beispiel>Text<Beispiel/>



Aufbau eines HTML Programms

<!DOCTYPE html>
<html>
 <body>
 </body>
</html>


Diesen Teil kann man als Anfänger eifach kopieren oder abtippen. Alles, was wir jetzt lernen werden, kommt zwischen <body> und </body>.



H Elemente

H ist die Abkürzung des englischen Wortes heading (Überschrift). Es gibt sechs verschiedene Grössen von Überschriften. Beispiel:


<h1>Das ist die grösste Überschrift</h1>
<h6>Das ist die kleinste Überschrift</h6>


Auf einer Website sähe das so aus:


Das ist die grösste Überschrift

Das ist die kleinste Überschrift


P Elemente

P ist die Abkürzung des englischen Wortes paragraph (Abschnitt/Absatz). Ps werden für alle Arten von Texen verwendet. Dieser Text ist auch in einem P Element. Beispiel:


<p>Das ist ein Absatz.</p>


Auf einer Website sähe das so aus:


Das ist ein Absatz.



Links

Viele Webseiten würden ohne Links gar nicht mehr funktionieren. Wenn man bei Google ein Suchergebnis auswählt, oder ein Youtube Video anklickt, benutzt man einen Link. Links werden nicht mit L oder H (hyperlink) abgekürzt. Man verwendet ein 'a'. Dort wo wir bisher den Text hingeschrieben haben, kommt jetzt das hin, was angezeigt werden soll. Wir müssen aber noch angeben, wo der Link hingehen soll. Dafür benutzt man HREF (Hyperlink REFerence). Beispiel:

<a href="http://eskottova.github.io"> Dieser Link führt zur Startseite. </a>


Dieser Link führt zur Startseite.


Links können auch in anderen Elementen sein:


<h4>Dieser
 <a href="http://eskottova.github.io"> Link </a>
ist in einer Überschrift. </h4>
<p>Dieser
 <a href="http://eskottova.github.io"> Link </a>
ist in einem Absatz.</p>


Dieser Link ist in einer Überschrift.

Dieser Link ist in einem Absatz.

Style

Bei den Links stand etwas im Anfangstag und nicht nur zwischen dem Anfangs- und Endtag. Das macht man immer, wenn man etwas schreiben will, das nicht als Text gesehen werden soll. Oft macht man das auch mit Farben. Es macht ja keinen Sinn, wenn man in den Text schreibt: "Meine Überschrift (eigentlich sollte sie rot sein)".

<p style="color:white;background-color:midnightblue;">Mein farbiger Absatz</p>


Mein farbiger Absatz

Um eine Farbe anzugeben, schreibt man entweder die hexadezimale oder rgb Zahl hin. Falls du diese noch nicht kennst, solltest du einfach den englischen Namen der Farbe ausschreiben.
